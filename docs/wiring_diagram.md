# TeleSol Wiring Diagram

## ESP32 Sensor Hub Pin Mapping
- HLK-LD2410 Radar: RX=16, TX=17
- VL53L0X ToF: SDA=21, SCL=22
- MPU6050 IMU: SDA=21, SCL=22
- INMP441 Mic: WS=25, SD=32, SCK=33
- SSD1306 OLED: SDA=21, SCL=22
- Status LED: 2

## ESP32-CAM Pin Mapping
- Flash LED: 4
- Built-in LED: 33

Refer to config.h for exact pin assignments.
